Задание:


Чем отличаются эти запросы:
Первый: SELECT * FROM a, b WHERE a.id=b.a_id;
Второй: SELECT * FROM a JOIN b ON a.id=b.a_id;
пришлите ответ текстом

Ответ:


По сути эти запросы отличаются скоростью их выполнения.
Результат же будет одинаковый. Этот SELECT всегда будет возвращать только те строки, которые совпадут значениями.
